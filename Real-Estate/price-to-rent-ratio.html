Price to Rent Ratio

Sep. 1, 2021

The price to rent ratio (P/R) is a well known litmus test for real estate deals. For example, if you could rent a $180,000 home for $1500/month, the P/R ratio is 120 months.

There is no quick, easy thumbrule for a good P/R ratio. One can see below that the median P/R ratio (black curve) has trended upward over the last 50 years, based on FRED data on median sales price of US homes (MSPUS) and a unitless rent index (CUUR0000SEHA).

The hypothesis of the marginal dividend model is that a target P/R ratio is set by combination of fiat monetary policy in parallel with a real-economy "resistance to becoming rundown". The Federal Reserve provides enough public data for a quick test, and the result is plotted below with a P/R index (black) normalized to the model (red) in December 2015. To improve credibility, US recessions are shown and the data starts the year Nixon suspended the gold standard.

Should I Buy or Rent?

The model provides a target of where fair market property prices are headed based on federal policy. With the real price ratio (black) normalized to the model (red) in Dec. 2015, it looks like property prices in April 2021 were quickly recovering to the target but not yet in a bubble.

Is the model valid? You can read more about the model in the series Should I Buy or Rent? The graph is based on the FRED datasets cited below, but I also relied on an anecdotal value of 180 months for the "run down resistance" Rreal based on my personal home expenses. You can test the MATLAB script yourself with my data scripting library econdata.m (its low quality, don't judge).

MATLAB Script

clear;
clc;
run('econdata.m'); % Functions for importing and manipulating data

mort30us = import_fred_csv('Data\MORTGAGE30US.csv');   % percent apr
mspus = import_fred_csv('Data\MSPUS.csv');             % dollars
rent_index = import_fred_csv('Data\CUUR0000SEHA.csv'); % unitless
recessions = import_fred_csv('Data\JHDUSRGDPBR.csv');  % 0 or 1

apr_range = months_range(mort30us, mort30us);
apr = stretch_align_index(mort30us, apr_range);
mpr = (1 + apr(2,:)./100).^(1/12);
Rmort = (1-mpr.^(-360)) ./ (1 - 1./mpr);
Rreal = 180;
Rth(1,:) = apr(1,:);
Rth(2,:) = (Rreal * Rmort)./(Rmort + Rreal);

norm_year = 2015;
norm_month = 12;
pr_range = months_range(mspus, rent_index);
Vmed = stretch_align_index(mspus, pr_range);
Irent = stretch_align_index(rent_index, pr_range);
ind1 = lookup(Rth(1,:), 12*norm_year + norm_month);
ind2 = lookup(Vmed(1,:), 12*norm_year + norm_month);
scale = Rth(2,ind1) / (Vmed(2,ind2)/(Irent(2,ind2)));
pr_index(1,:) = pr_range(1) : pr_range(2);
pr_index(2,:) = scale * Vmed(2,:)./Irent(2,:);

plot(pr_index(1,:)/12, pr_index(2,:), 'k');
hold on;
plot(Rth(1,:)/12, Rth(2,:), 'r');
plot(recessions(1,:)/12, 70*recessions(2,:)+40);
title("Median Sales Price to Rent Index, normalized to Dec. 2015");
h = ["Median Sales P/R Index"; "Marginal Dividend P/R Ratio"; "US Recessions"];
legend(h, "location","southeast");
ylabel("months");
axis([1968, 2023,40,110]);
hold off;
