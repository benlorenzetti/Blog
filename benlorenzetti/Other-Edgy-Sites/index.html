  • Essential C

    An incredibly compact and complete text on C programming from Stanford. I highly recommend it if you have programmed C before but cannot remember syntax, libraries, etc. It is so brief that I actually learned some features I never would have reached in other texts due to laziness. Its brevity makes it more useful than Google for lookups.
