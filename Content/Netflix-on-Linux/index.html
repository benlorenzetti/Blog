Watching Netflix on Linux

Netflix requires that you have the Silverlight plug-in from Microsoft to stream videos. Silverlight is free but there is no Linux version. Fortunately, there is a workaround: you can run the Windows version using Wine.

  1. Install Wine

    Wine is a program that recreates the Windows environment on Linux. What this means is that you can run Windows programs, such as Silverlight. It is not super clear what Wine is an acronym for. Maybe WINdows Environment, maybe WINdows Emulator, maybe Windows Is Not an Emulator, or maybe it's Wine Is Not inEbriating. Regardless, it is easy to install with the package managers. For Red Hat, SUSE, or Fedora, open a terminal and use yum. If you are using Debian or Ubuntu, replace "yum" with "apt-get":

    $ sudo yum install wine -y

  2. Download a Windows version of your favorite web browser

    You need to go online and download an installer for a web browser. It is important that you download a version that is for Windows (i.e. has a .exe file extension). Sometimes this is difficult because the homepages for Chrome, IE, and Firefox will detect your Linux OS and automatically direct you to the Linux versions. If you are having trouble, you can find Firefox for Windows at https://www.mozilla.org/en-US/firefox/all.

  3. Run the .exe installer with wine

    First, make a new directory for the installation.

    $ cd /home/usernamehere
    $ mkdir Windows_Programs

    For new Linux users:
    In the change directory (cd) command, you should replace "usernamehere" with the username you logged in with. You can use "$ hostname" to get your username if you are unsure.

    Next, use the following commands to move into the Downloads directory and run the .exe installer.

    $ cd /home/usernamehere/Downloads
    $ ls
    some_files    Firefox Setup 24.0.exe    some_more_files
    one_more_file
    $ sudo wine Firefox\ Setup\ 24.0.exe

    For new Linux users:
    By default, downloads from the internet are stored in the Downloads folder. If you chose "save as" and downloaded to a different folder, you will need to navigate to that folder instead of Downloads. The list command (ls) shows you the files in the current directory. Look for the .exe file which you just downloaded. When you type the wine command, you only need to type the first several letters of this filename. Simply hit 'tab' and Linux will automatically complete the name.

    Once the installer has loaded, choose a custom install so you can select the destination folder. Select the new folder you created earlier. Hit next many times to finish the installation.

  4. Run the web browser with Wine

    Change directories to the installation folder and determine the name of the executable program. Then run the executable using Wine.

    $ cd /home/usernamehere/Windows_Programs
    $ ls
    firefox.exe
    $ wine firefox.exe
