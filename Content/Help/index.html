FTP procedure

The server probably has firewall protection, which must temporarily be stopped. Open a terminal and then SSH into the server.

$ ssh -i publicKey.pem username@domainname.com
$ sudo service iptables stop

Open another terminal to do the file transfer. Make sure to restart iptables after completing all the file transfers.

$ ftp -p domainname.com
Name: enterUsernameCreatedForFTP
Password:

ftp> put put /localMachinePathway/fileToTransfer saveAsFilename
ftp> get remoteMachineFilename /localMachinePathway/fileToDownload
ftp> quit

$ rm filename
