Economic Data Scripts

This page serves as a directory for various scripts I use for plotting economic data.

The MATLAB data format used throughout is [2×n] matrices because it makes plotting ok. The first row is usually months since December of 1 B.C. and the second row holds the data itself for that matrix.

Federal Reserve Economic Data (FRED)

The FRED online database has many authoritative datasets and provides free, conveniant, and easy to use downloading to CSV formatted data. Importing these CSV files into the MATLAB data format is very simple.

% "run-to-load" the data importing and manipulation functions run('econdata.m'); a = import_fred_csv("FREDDATA.csv"); plot(a(1,:)/12, a(2,:));

Stock Market Data from Irrational Exuberance

Professor Robert Shiller makes his research data available to the public at his website Online Data. Again econdata.m provides an importing function for this data (but resave the excel file as a CSV file first).

% Row Organization: % (1) Month, (2) S&P Price, (3) Dividend, (4) Earnings, (5) CPI, (6) GS10 large_table = import_shiller_csv("excel-saved-to-csv.csv"); cpi = shiller_series(large_table); plot(cpi(1,:)/12, cpi(2,:));

Freddie Mac Single-Family Loan-Level Dataset

This dataset is also free to the public but it has terms and conditions, in particular I make sure it is never accidentally post it externally except for research related graphs that use derived medians/averages to test my equations and hypotheses.

Importing functions in freddie-mac-origination-data.h and scripts like down-payment.c convert this large amount of data to usable series for short scripts in MATLAB/Octave.

The function script real_estate_eqs.m contains useful equations derived in the mortgage calculator series.
